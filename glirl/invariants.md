3\. Logical Invariants

Logical invariants ensure that GLIRL remains a deterministic logic system.



3.1 Irreducibility of NOT

The NOT operator is primitive and cannot be decomposed.



Invariant:



Code

NOT is irreducible.

Violation would collapse the Pure‑NOT substrate.



3.2 Deterministic Contraction

Every GLIRL expression must reduce to a single canonical form.



Invariant:



Code

∀ expr: contraction(expr) → unique\_fixed\_point

This ensures reproducibility across the civilisation.



3.3 No Hidden Operators

All operators must expand into Pure‑NOT + PAIR.



Invariant:



Code

∀ op: expand(op) ∈ {NOT, PAIR}

This prevents semantic drift.



3.4 Kernel Consistency

The evaluation kernel (AND or NOR) must remain consistent within a reasoning context.



Invariant:



Code

kernel(expr₁) = kernel(expr₂)

Mixed kernels are forbidden unless explicitly lifted.



4\. Structural Invariants

Structural invariants ensure that GLIRL expressions maintain structural coherence.



4.1 PAIR Must Be Binary

PAIR must always contain exactly two children.



Invariant:



Code

PAIR(a, b) is valid

PAIR(a) or PAIR(a, b, c) is invalid

4.2 No Cyclic ASTs

GLIRL ASTs must be acyclic.



Invariant:



Code

expr cannot contain itself as a descendant

This prevents infinite regress.



4.3 Canonical Ordering

PAIR children must be ordered deterministically.



Invariant:



Code

PAIR(a, b) canonical ordering rule applies

This ensures reproducible hashing and identity.



4.4 No Implicit State

GLIRL expressions cannot depend on external state.



Invariant:



Code

expr evaluation depends only on expr

This is essential for zero‑trust reproducibility.

5.4 Figment Continuity

6\. Cross‑Layer Invariants

Cross‑layer invariants ensure that reasoning remains coherent across the four layers of the Civilisation‑Stack.



6.1 No Upward Violation

Higher layers cannot violate lower‑layer invariants.

6.3 GROUND Must Preserve Invariants

6.4 Cross‑Layer Alignment

Cross‑layer reasoning must maintain structural alignment.



Invariant:



Code
