2\. What a Witness Is

A Witness is a structural entity that:



observes an event



validates invariants



records the transition



produces an attestation



Formally:



Code

WITNESS(event, context) → ATTESTATION

A witness is not a judge.

A witness is a verifier.



Witnesses operate using:



GLIRL invariants



Pure‑NOT evaluation



cross‑layer alignment checks



narrative continuity operators

Code
